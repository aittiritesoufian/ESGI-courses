# JQuery

### Appeler un objet JQuery:

```JS
JQuery(document).ready(function(){
	/* code jQuery */
});
```
OU
```JS
$(document).ready(function(){
	/* code jQuery */
});
```

### fonctions importantes:

html(): récupère le contenu d'un élément/de l'éditer

```JS
$('div').html('nouveau content');
```
